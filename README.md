# 🌙 lofi machine

Rule-based lo-fi beat generator with controlled randomness. Built with Streamlit.

## Features

- **Tempo & Structure**: BPM, bar count, reproducible seeds
- **Harmony**: 7 scale presets, 5 chord progressions
- **Rhythm**: 5 drum patterns, adjustable swing
- **Mixer**: Toggle and volume-control each layer independently
- **Character**: Melody density, note range, detune, vinyl noise, warm pad
- **Effects**: Stereo delay with independent L/R timing
- **Stereo**: Per-track panning

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy Free on Streamlit Community Cloud

1. Push this folder to a GitHub repo
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect your repo → select `app.py`
4. Deploy

That's it — free hosting with a public URL.
